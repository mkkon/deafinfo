情報収集

「情報を正しく収集する」ことを目的に、情報が示していることを正しく読み取る力、情報を収集する力を習得する。
前者は情報の価値・意味を判断するために、後者は効率よく情報を収集するために必要な力となる。

なお、本資料の講義資料（別タブ閲覧推奨）も併せて公開する。
