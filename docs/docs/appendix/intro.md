# 謝辞
本原稿の執筆に当たり、筆者の周囲の人々から多くの協力を頂いた。この場を借りて感謝を表す。
本資料の執筆には[gitbook](https://www.gitbook.com/)を用いた。
また、本資料は[『当事者に学ぶ視覚・聴覚障がい者のセルフアドボカシープログラム』](https://self-advocacy.themedia.jp/)の一環として作成されている。
